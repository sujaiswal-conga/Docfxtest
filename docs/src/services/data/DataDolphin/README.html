DataDolphin (:dolphin:)

Helps in interacting with RLS platform for data gathering, reading / writing templates and documents.

Objective

The purpose of this service is to interact with RLS Platform for following duties.

  • Analysis template(s) and prepare the data needed for them.
  • Fecth the templates needed for merge operation.
  • Read / write documents to RLS platform.

Messages

Incoming Messages

DataGatherMessage

This message is received by the service when you need to prepare the data that is needed for merge for a template.

DocumentUploadMessage

This message is received by the the service when you need to upload any document to RLS.

GetDocumentMessage

This message is received by the the service when you need to fetch a document from document store.

GetDocumentsMessage

This message is received by the the service when you need to fetch many documents from document store.

Outgoing Messages

DataGatherCompleteMessage

This message is published from the service after the data gathering is completed for merge operation.

DocumentUploadCompleteMessage

This message is published from the service after the document is uploaded.

GetDocumentCompleteMessage

This message is published from the service after the document is fetched.

GetDocumentsCompleteMessage

This message is published from the service after the documents are fetched.
