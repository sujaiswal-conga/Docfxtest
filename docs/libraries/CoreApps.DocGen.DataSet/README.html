Conga.CoreApps.DocGen.DataSet

Description

This is a C# library that extends the built-in DataSet (SmartSet) and adds additional capabilities like retrieving data using XPath like syntax, binds custom metadata to tables and columns etc.

Installation

dotnet add package Conga.CoreApps.DocGen.DataSet

SmartSet Methods

void Fill(string schemaAndDataXml) - Fills the smart set with schema and data.

void Fill(System.Data.DataSet dataSet) - Fills the smartset from dataset.

void Fill(IList schemas, IDictionary<string, List<Dictionary<string, dynamic>>> data) - Fills the smartset with schema and data.

object Get(string path, object source = null) - Returns data from the passed path. The path looks like XPath for ex. "/Apttus__APTS_Agreement__c/Apttus__Account__c/account_name".

T Get(string path, object source = null) - Returns data from the passed path.

IList Query(string path, string filters = null, string sortArgs = null, object source = null) - Queries collection from the passed path applying filtering and sorting.

string Render(string path, string format = null, object source = null) - Retrieved data from the passed path and then formats the retrieve data based on the passed format or from the default value configured for the data type.

string Format(object data, string format) - Formats data in the passed format.

EntityMetadata GetEntityMetadata(string name) - Returns entity metadata that is tied with the passed table name.

void AddEntityMetadata(EntityMetadata entityMetadata) - Creates new table from the passed entity metadata information and adds to the dataset.

FieldMetadata GetFieldMetadata(string entityName, string fieldName) - Returns field metadata from the passed table and column name.

void AddData(string entityName, params object[] data) - Adds new row for the passed entity or table.

T GetMetadata(string path, object source = null) where T: Metadata - Returns the bound entity or field metadata from the passed path.
