Table of Contents

Conga.CoreApps.DocGen.Base

Provides abstractions for all DocGen libraries.

BaseDocument

The abstract base class for all types of documents (Word, PDF etc.). Provides methods to load and save documents from file, stream etc. It also contains some basic properties like Type, Author etc.

ITemplate

Represents a generic template of any document type.

ITemplateMergeEngine and ITemplateMergeEngine<T, U>

Represents a merge engine that merges the passed template with different data sources and generates document. All merge engines should implement this interface directly or indirectly.

BaseTemplateMergeEngine<T, U>

The abstract base class for all merge engines.

IMergeCallback

Represents a callback that runs before or / and after the merge operation.

ITemplateTransformer and ITemplateTransformer<T, U>

Represents a component that transforms one type of template to another.

MergeEnginesRegistry

Registry for all the merge engines.

TemplateTransformersRegistry

Registry for all the template transformers.

ITemplateDataSource

Represents the data source for a template. All template data sources should implement this interface directly or indirectly.

IDocumentAction and IDocumentAction<T>

Represents an action that can be performed on any kind of a document. These actions are purely CPU bound operation and should not involve any I/O.

DocumentAction<T>

The abstract class for all document actions.

IActionRunner

Runs queue of actions on single or multiple documents.

Support Classes

  • Base64Helper, GZip, Serializer etc.
