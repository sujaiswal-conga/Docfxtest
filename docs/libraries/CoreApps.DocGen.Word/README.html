Conga.CoreApps.DocGen.Word

House of all class libraries for processing MS Word documents. Currently all the modules are bundled into a single package called Conga.CoreApps.DocGen.Word.

Installation

dotnet add package Conga.CoreApps.DocGen.Word

List of modules

Core

Contains core class libraries that contains some common classes and abstractions needed for processing MS word documents.

  • CoreApps.DocGen.Word.Core => Contains the core classes for Word processing like WordDocument, IWordTemplateMergeEngine` etc.

Templates

Contains class libararies that contains object models for different templates built on MS Word documents.

  • CoreApps.DocGen.Word.Aspose - Contains classes for Aspose Linq Engine based templates.
  • CoreApps.DocGen.Word.XAuthor - Contains classes for XAuthor templates. XAuthor is a home-grown template solution.
  • CoreApps.DocGen.Word.Native - Contains classes to represent a generic template that contains pure classes not tied with Word or any format.
  • CoreApps.DocGen.Word.Web - Contains classes to represent a Word template in HTML.

Transformers

Contains class libararies that transforms one template type to another.

  • CoreApps.DocGen.Word.XAuthor2Aspose - Contains classes that helps to transform an XAuthor template to Aspose template.
  • CoreApps.DocGen.Word.XAuthor2Native - Contains classes that helps to transform an XAuthor template to Native template.
  • CoreApps.DocGen.Word.Native2Aspose - Contains classes that helps to transform a Native template to Aspose template.
  • CoreApps.DocGen.Word.Native2Web - Contains classes that helps to transform a Native template to HTML template.
  • CoreApps.DocGen.Word.Web2Native - Contains classes that helps to transform a Web template to Native template.

Engines

Contains merge engines. Currently we've two merge engines and both are based out of Aspose library.

CoreApps.DocGen.Word.AsposeLinqEngine - Advanced merge engine for Aspose Linq Engine Based Templates. CoreApps.DocGen.Word.AsposeMergeEngine - Advanced merge engine for Aspose Merge Field Based Templates.

Export

Contains the export library Conga.CoreApps.DocGen.Word that references all the other ones.
