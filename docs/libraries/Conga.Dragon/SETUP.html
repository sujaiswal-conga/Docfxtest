Setting up Dragon for development

Messaging

To simplify development and testing the messaging modules we've created two console test apps: MessagePublisherTest and MessageSubscriberTest. First you need to create the queues and a sample topic by running the powershell scripts placed under the "scripts" folder at root.

You need to run the "publisher_queue_setup.ps1" and "subscriber_queue_setup.ps1" to create the publisher, subscriber queues and a sample topic.

You need to set both the console apps as start projects and running them first publisher will send a sample message to subscriber and then the subscriber sends back a reply message.
