Conga.Dragon (:dragon_face:)

Framework for distributed programming. Dragon helps to simplify programming when you have a distributed system made-up of microservices that communicates each other through messaging.

Provides support for Messaging, Workflow, Transaction Management, Temporary Data Storage and more.

Quick Setup

Install Conga.Dragon.Domain in your Domain and Application projects of your microservices.

Install-Package Conga.Dragon.Domain

The Conga.Dragon.Domain module helps to develop core business domain using DDD principles and Hexagonal architecture. It also provides abstractions for CQRS commands and handlers.

Install Conga.Dragon in your Infrastructure project.

Install-Package Conga.Dragon

Conga.Dragon simplifies setup by automatically configuring messaging, transaction and caching modules to your project.

Install Conga.Dragon.Web.Extensions to your web application and Conga.Dragon.Worker.Extensions to your worker.

For example, in your web and worker projects you can configure Dragon as below.

In Web

public class Startup : DragonWebStartup
{
	public Startup(IWebHostEnvironment environment, IConfiguration configuration)
	{
		Init(configuration, new WebAppConfiguration
		{
			ServicePrefix = "docgen",
			DragonConfig = new DragonConfig(),
			DragonStartupParams = new DragonStartupParams(new MessagingSetupOptions
			{
				Assemblies = new Assembly[] 
				{ 
					typeof(DocumentWorkflowRequestMessage).Assembly
				}
			})
		});
	}

	protected override void ConfigureAppContainer(IServiceCollection services)
	{
		base.ConfigureAppContainer(services);
	}
}

In Worker

public class Startup : DragonWorkerStartup
{
    public Startup(IConfiguration configuration, IWebHostEnvironment environment)
    {
        Init(configuration, new WorkerConfiguration
        {
            DragonConfig = new DragonConfig
            {
                ConfigureSubscriber = true
            },
            DragonStartupParams = new DragonStartupParams(new MessagingSetupOptions
            {
                Assemblies = new Assembly[] 
                { 
                    typeof(DocumentWorkflowRequestMessage).Assembly,
                    typeof(DocumentWorkflowRequestMessageHandler).Assembly
                }
            })
        });
    }

    protected override void ConfigureAppServices(IServiceCollection services)
    {
        base.ConfigureAppServices(services);
        services.ConfigureWorkerServices(Configuration);
    }

    protected override void ConfigureAppContainer(IServiceCollection services)
    {
        base.ConfigureAppContainer(services);
        ObjectDBClientRegistration.AddObjectDBClientForWorker(services, Configuration);
        UserManagementClientRegistration.AddUserManagementClientForWorker(services, Configuration);
    }
}
